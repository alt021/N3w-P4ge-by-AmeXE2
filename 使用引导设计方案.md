# 总体目的

为初次使用本扩展的用户提供一个功能说明，和必要的提示

# 交互和视觉设计

以弹出式对话框的设计，初次安装扩展并打开此页面时自动弹出，且完成引导前不可关闭。
对话框以圆角矩形的方式呈现，四周添加轻微阴影营造悬浮感，并且添加黑色半透明蒙版

# 引导流程

大致上分为四个步骤

1. 对话框中间以黑色加粗大标题书写：欢迎使用AmeXE2的新标签页，下面写副标题“一个现代 极简 纯净的浏览器主页”
2. 扩展使用许可，和特别提醒。顶部居中书写大标题："开源说明与使用许可"。副标题："请仔细阅读相关说明"。下面通过可上下滚动的文本框先后呈现以下的内容

> ### 扩展说明
> 1. 本项目为Vibe Coding产物，仅供作者自用。代码质量较差，如有需要请自行重构或修改。**
> 2. 本扩展完全离线运行，不收集、上传或存储任何有关于您的数据或信息**
> 3. 为实现快捷方式功能，本扩展需要您的历史记录和书签的访问权限，数据仅在本地存储，不会被上传或收集**
>
> ### 开源说明
>
> \[此处呈现完整的MIT许可证内容\]

3. 使用方法引导，顶部标题书写点按右键打开设置菜单，下面放置我已经提前放在项目根目录下的scrshot.png，对图片做圆角处理
4. 引导结束页，中间放一个礼花的emoji，下方标题写欢迎使用

每个页面底部都放置圆角矩形的按钮，四个页面中的按钮文本分别为“开始”“同意”“继续”“完成”
